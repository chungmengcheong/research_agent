# Source Registry

This file lists the named sources used by this repo and how they should be treated.

For each source, specify:
- **Type**: Reported news, Primary source, or Curated interpretive source
- **Quality**: relative confidence in signal quality
- **Used for**: Daily notes, Synthesis, or both
- **Notes**: how to interpret and weight the source

Where not explicitly stated, treat a newly added source conservatively:
- Type: Curated interpretive source
- Quality: Medium
- Used for: Synthesis only

## Reported news

- **Techmeme**
  - Type: Reported news
  - Quality: High
  - Used for: Daily notes
  - Notes: Use the complete home page as the default daily source for observed developments. 

## Primary sources
